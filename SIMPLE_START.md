# ✅ Simple Start Guide

## Your Data is Safe!

Your `team_data.csv` file is still there with all your team information. We just need to import it into the app.

## Quick Start

### 1. Start the App
```bash
npm start
```

### 2. Import Your Data
When the app opens, you'll see a "Load Team Data" screen:
- Click **📂 Load Data File**
- Select your `team_data.csv` file
- Your team will appear!

That's it! All your data (employees, training, objectives, tasks) will be loaded.

## If Something Goes Wrong

1. **Buttons don't work?**
   - Refresh the page (F5)
   - Close and restart the app

2. **Data not importing?**
   - Make sure `team_data.csv` is in the project folder
   - Check that the file format is correct

3. **App won't start?**
   - Run `npm start` again
   - Check Command Prompt for errors
